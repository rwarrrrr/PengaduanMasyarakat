## Pengaduan Masyarakat
Aplikasi Berbasis Website Sederhana Tentang Pengaduan Masyarakat Menggunakan Laravel + Mysql

## Fitur Admin
- Login
- Register
- Logout
- Verifikasi Dan Validasi
- Memberikan Tanggapan
- Generate Laporan To PDF

## Fitur Petugas
- Login
- Logout
- Verifikasi Dan Validasi
- Memberikan Tanggapan 

## Fitur Masyarakat
- Login
- Register
- Logout
- Membuat Pengaduan
